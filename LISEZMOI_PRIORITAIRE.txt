❌ ERREURS COURANTES & SOLUTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Impossible de se connecter à la base de données"
✅ Solution: Vérifier SQL Server/LocalDB, modifier connection string

❌ "L'avatar Simli n'apparaît pas"
✅ Solution: Ouvrir F12, chercher erreurs, vérifier URL

❌ "La reconnaissance vocale ne fonctionne pas"
✅ Solution: Autoriser l'accès au microphone, utiliser Chrome/Edge

❌ "OpenAI API error"
✅ Solution: Vérifier clé API, vérifier quota OpenAI

Lire le GUIDE_INSTALLATION_COMPLETE.txt pour TOUTES les solutions!
