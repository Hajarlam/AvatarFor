❌ ERREUR: "The API call to OpenAI failed"

✅ SOLUTION:
   1. Vérifier que l'Internet fonctionne
   2. Vérifier la clé API OpenAI dans appsettings.json
   3. Vérifier le quota OpenAI (possibilité de dépassement)
   4. Redémarrer l'application: Ctrl+C puis dotnet run


❌ ERREUR: "Cannot connect to database"

✅ SOLUTION:
   1. Vérifier que SQL Server/LocalDB fonctionne
   2. Dans Visual Studio: View → SQL Server Object Explorer
   3. Ou démarrer SQL Server Express
   4. Modifier appsettings.json (connection string):
      "Server=(localdb)\\mssqllocaldb;Database=LMS_DB;"


❌ ERREUR: "L'avatar Simli n'apparaît pas"

✅ SOLUTION:
   1. Ouvrir F12 (Console navigateur)
   2. Chercher les erreurs
   3. Vérifier: "✅ Avatar Simli chargé"
   4. Vérifier la session URL dans AvatarChat.cshtml
   5. Rafraîchir la page: Ctrl+F5 (cache)


❌ ERREUR: "Fichier CSS/JS non trouvé"

✅ SOLUTION:
   1. Les CSS/JS sont du CDN (Bootstrap, Font Awesome)
   2. Pas d'installation locale nécessaire
   3. Vérifier la connexion Internet
   4. Vérifier les URLs dans _Layout.cshtml


❌ ERREUR: "Page Not Found (404)"

✅ SOLUTION:
   1. Vérifier la route: /Home/Dashboard
   2. Vérifier le nom des fichiers Razor (.cshtml)
   3. Vérifier la casse (Linux sensible à la casse)
   4. Redémarrer: F5 (Visual Studio)


❌ ERREUR: "Session utilisateur perdue"

✅ SOLUTION:
   1. Session timeout défini à 2 heures
   2. Si Logout spontané: reconnexion automatique
   3. Vérifier appsettings: "IdleTimeout = TimeSpan.FromHours(2)"
